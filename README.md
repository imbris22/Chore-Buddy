## Running the implementation prototype
Follow the steps below to run the Chore Buddy app and reproduce the results shown in video demonstration.

### 1. Download and open the repository
* Clone or download this repository from GitHub.
* Open the project folder in Visual Studio Code (VS Code).

### 2. Navigate to the project directory
* In the VS Code terminal, run: <code>cd chore-buddy</code>.

### 3. Install dependencies
* Then run: <code>npm install</code>.

### 4. Start the development server
* Run the following command: <code>npx expo start -c</code>.

### 5. Set up your mobile device
* Install the <b>Expo Go</b> app on your Android or iOS phone device.

### 6. Launch the app
* Scan the QR code displayed in your terminal using your device's camera.
* Open the project in <b>Expo Go</b> when prompted.

### 7. Interact with the prototype
* Once the app loads, click on the icons to navigate the app.
